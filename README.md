# Implementação-A*-Grafos
## Como Executar:
- Instale a biblioteca PyGame.
- Rode o AEstrela.py normalmente usando python
- Use o botão esquerdo do mouse para definir o inicio, o fim e os obstáculos.
- Use o botão direito para apagar um nó.
- Use "C" para limpar a tela.
- Quando estiver satisfeito use o espaço para iniciar o algoritmo.
- Observe.
